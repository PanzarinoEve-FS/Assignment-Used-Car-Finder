# Assignment-Used-Car-Finder

In this week's activity, you are tasked with building a new ES6/HTML/CSS application that will run in your browser. The purpose of this application is to find used cars based on their year, make, and model—similar to the functionality found on websites like the Kelley Blue Book.



**The Design**
You do not need to create a custom design for this activity. The design provided below comes with its assets, including background images and colors. It is your responsibility to write the CSS for the provided design from scratch.


Now that you have the design composition, the next asset on the list is the style guide. The style guide’s purpose is to show front-end developers the fonts, colors, and other details used in the design composition. It is typically presented in an easy-to-digest format, using elements like color swatches and typography examples.

I have attached a working example that I built to help all of my students fully understand the requirements of this application. You can see what happens when I select a car's year, make, and model.

Note that both the make and model fields should be grayed out until the previous selection has been made. For instance, the user shouldn't be able to select a model until they have first chosen the year and make. Once the user has selected all three traits, the full details of the car should be printed in the console (as shown on the right in my video demonstration).

Webpack Project Template

The last thing I will provide is a project template for running your application. It uses Node.js and Webpack to compile ES6 into older, compatible JavaScript that can run in any web browser, even if it doesn't support ES6 yet. This process is called transpilation.

All you need to do is download the .zip file and extract it to an empty folder on your machine. Then, navigate to the folder in your terminal and run the following command to download and install the Node modules:
Bash
npm install
npm run start


After running "npm install," you will need to run "npm run start" to start your project. If everything is set up correctly, it will open a new browser window displaying your application.
